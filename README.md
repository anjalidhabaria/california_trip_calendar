# California Trip Calendar

## Project Description
The California Trip Calendar is a simple application designed to help users plan and organize their trips to California. Users can easily create trips, add events, and manage their itineraries, ensuring a smooth and enjoyable experience.

## Features
- Create and manage trips
- Add events to each trip
- View a calendar of scheduled events
- Simple and intuitive user interface
